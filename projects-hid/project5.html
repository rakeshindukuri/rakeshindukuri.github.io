Project 5: Cancer Analysis using Machine Learning

Feb 2023 - April 2023

Personal Project

Project Details:

This project is an analysis of the Breast Cancer Wisconsin (Diagnostic) dataset, which includes measurements of various features of breast tissue, as well as a diagnosis of either benign or malignant. The project begins with loading the data and performing visualizations to understand the relationships between the features and the diagnosis. The data is then standardized, and four machine learning algorithms (Logistic Regression, Support Vector Machines, Decision Trees, and Random Forest) are applied to the dataset, using different combinations of feature sets. The Random Forest algorithm achieved the highest accuracy in predicting diagnosis. The project then examines feature importance for the Random Forest and Support Vector Machines algorithms, and performs further analysis on the worst-performing features. Principal Component Analysis is also used for feature extraction, and Gradient Boosting and XGBoost algorithms are explored. Finally, an Artificial Neural Network (ANN) is trained to predict diagnosis, achieving high accuracy after implementing feature scaling. Overall, the project showcases the importance of feature selection and algorithm choice in achieving accurate predictions in medical diagnosis.
